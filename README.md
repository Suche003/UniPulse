# UniPulse

UniPulse is a MERN-based university event management system.

## Team Modules
- User Participation + Super Admin: completed by project lead
- Event Management: Member 01
- Stall Management: Member 02
- Sponsor Management: Member 03

## Tech Stack
- Frontend: React + Vite
- Backend: Node.js + Express
- Database: MongoDB Atlas
